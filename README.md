## ⚔️ The Mission

Rock, Paper, Scissors — built from scratch in C as a fundamentals exercise.

No game libraries, no shortcuts. Just `scanf`, comparisons, and a fight against the input buffer.
Part of the broader "learning C from the ground up" scratchpad, prepping for KMITL bootcamp.

## 🎮 How to Play (Planned)

Human vs. CPU. One round, three possible moves:

| Input | Move     |
|-------|----------|
| `r`   | Rock     |
| `p`   | Paper    |
| `s`   | Scissors |

Type a letter, hit enter, see who wins. That's the whole game.
